## System Requirements
Pastikan perangkat Anda memenuhi persyaratan berikut:
- Node.js v18.12.0
- npm v8.19.2
- Vue 2

## Instalasi dan Menjalankan Project
1. Download file project.
2. Buka terminal
3. Masuk ke folder project:
   ```bash
   cd nama-project
4. Install semua dependencies:
   ```bash
   npm install
5. Jalankan project:
   ```bash
   npm run serve
6. Buka browser dan akses:
   ```bash
   http://localhost:8080/
